# Virtual Jo: The Handy MCSP Helper

This is the repository for my senior project.

## Installation

Run `npm install`

## Commands

Run commands in the form `npm run <command>`

```
deploy - generates the required files and uploads the script to Google Cloud Functions

makecourses - generates the files needed for courses

testdeploy- uploads the project files in its current state.

menusaver - downloads the menu files
```
